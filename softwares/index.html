In the digital age where tools shape the way we think and work, my software choices reflect precision, curiosity, and a focus on reproducible research. Every project I undertake is supported by a curated suite of tools that accelerate productivity and improve reliability. This page documents the ecosystem that powers my research, development, writing, teaching, and content creation.

Machine Learning & AI Development

Deep Learning Frameworks

  • PyTorch – Primary framework for deep learning research and model development.
  • TensorFlow – Alternative framework for machine learning projects.
  • PyTorch Lightning – High-level framework for structured, scalable training on top of PyTorch.
  • Hugging Face Transformers – Library for state-of-the-art transformer-based NLP models.

Python Ecosystem

  • Python 3.x – Core programming language for ML, scripting, and automation.
  • Jupyter Notebook – Interactive computational environment for experiments and prototypes.
  • Google Colab – Cloud-based environment for Python and GPU-accelerated ML.

Experiment Configuration & Versioning

  • Hydra – Configuration framework for managing complex experiment setups.
  • Data Version Control (DVC) – Data version control and pipeline automation for ML projects.
  • TensorBoard – Visualization toolkit for training metrics and model graphs.

Development Environment

Code Editors & IDEs

  • Visual Studio Code (VS Code) – Primary editor for coding, writing, and note-taking.
  • RStudio – Integrated development environment for R.
  • Doom Emacs – Highly configurable Emacs configuration for advanced text editing.
  • Notepad++ – Lightweight editor for quick text editing on Windows.

Environment & Project Management

  • Conda / Micromamba – Environment and package management for Python and data science.
  • Lightning-Hydra Template – Structured ML project template combining PyTorch Lightning, Hydra, and best practices.

Terminal & Shell

  • Windows Terminal – Modern terminal solution on Windows.
  • PowerShell – Scripting and task automation on Windows.
  • Windows Subsystem for Linux (WSL2) – Linux environment on Windows, primarily using Ubuntu.

Research & Academic Writing

Literature Management

  • Zotero + BetterBibTeX – Reference management, citation keys, and BibTeX/CSL JSON export.
  • Google Scholar – Literature search and citation tracking.
  • ORCID, Scopus, ResearchGate – Researcher profiles and academic networking.

Document Preparation

  • Quarto – Primary tool for presentations, reports, thesis chapters, and documentation.
  • LaTeX – Typesetting for technical documents, including custom classes like ManitThesis.cls and MANIT Lab Manual.
  • Pandoc – Universal document converter behind many of my workflows.
  • Markdown – Lightweight markup language for notes and documentation.
  • TinyTeX / pdflatex – Minimal TeX distribution and PDF generation tooling.

Writing & Note-Taking

  • VS Code – Primary environment for writing Quarto/Markdown documents.
  • Obsidian – Knowledge management and personal note-taking using Markdown.

Diagramming & Graphics for Papers

  • TikZ/PGF – LaTeX-based diagrams for high-quality figures.
  • Mermaid – Text-based diagrams and flowcharts embedded in Markdown/Quarto.
  • draw.io (diagrams.net) – Browser-based diagramming tool.
  • Inkscape – Vector graphics editor for technical diagrams and figures.

Statistical Computing with R

Core R Stack

  • R – Statistical computing and data analysis.
  • tidyverse – Collection of R packages (e.g., dplyr, ggplot2, tidyr, purrr) for data science.
  • rmarkdown – Dynamic documents using R and Markdown.
  • pagedown – HTML-based paged documents, including academic CVs.

Typical Use Cases

  • Automation for Systematic Literature Reviews (SLRs).
  • CV generation via an Excel → R → pagedown pipeline.
  • Statistical analysis, visualization, and reproducible reports.

Data Visualization & Graphics

Data Visualization Libraries

  • Matplotlib – Core plotting library for Python.
  • Seaborn – Statistical data visualization built on top of Matplotlib.
  • ggplot2 – Grammar of graphics implementation in R.
  • Plotly – Interactive graphing library for Python and R.
  • Tableau – Business intelligence and dashboarding.

Image & Design Tools

  • GIMP – Raster graphics and photo editing.
  • LibreOffice Draw – Diagrams and sketches.
  • Microsoft PowerPoint – Slide design and visual content creation.

High-Performance Computing

Cluster & Remote Computing

  • Portable Batch System (PBS) – Job scheduling on HPC clusters using commands like qsub and qstat.
  • SSH / SCP – Remote server access and secure file transfer from the terminal.

Monitoring & Hardware Tools

  • htop – Interactive process and resource monitor in terminal.
  • nvidia-smi – GPU monitoring and management for NVIDIA GPUs.

Version Control & DevOps

  • Git – Distributed version control system.
  • GitHub – Code hosting, issues, and collaboration.
  • GitHub Actions – CI/CD automation for testing, deployment, and Quarto publishing.
  • Data Version Control (DVC) – Dataset and model versioning, experiment tracking, and pipelines.

Productivity & System Management

System & Configuration Management

  • Windows Package Manager (winget) – Package manager for installing and updating software on Windows.
  • chezmoi – Cross-machine dotfiles management for consistent development environments.

Productivity & Everyday Tools

  • Raindrop.io – Bookmark and web resource manager.
  • Microsoft Edge – Primary web browser.
  • Espanso – Cross-platform text expansion.
  • Slack / Microsoft Teams – Communication and collaboration platforms.

Cloud Storage

  • Google Drive – Cloud storage, backup, and file sharing.

Office Suites

Open Source Projects

I actively develop and maintain several open-source projects that support reproducible research and academic workflows. All projects are freely available on GitHub.

Quarto Extensions

MANIT Thesis Quarto Extension (manit-thesis)

Quarto extension that applies MANIT-approved thesis formatting through a reusable template for fully reproducible dissertation authoring. Supports LaTeX, R, and Pandoc for professional academic document generation.

GitHub Repository

MANIT Pre-Thesis Quarto Extension (manit-pre-thesis)

Quarto document template for Pre-PhD pre-thesis formatting with automated LaTeX styling, compliant with MANIT pre-thesis submission guidelines.

GitHub Repository

Contributing

  • Submit pull requests with enhancements.

  • Share your use cases and feedback to help improve future releases.

Most projects are licensed under permissive open-source licenses (typically MIT or GPL) to ensure reusability and benefit to the wider research community.
